Để tạo biến bạn chỉ cần dùng 2 dấu hai chấm là được. Ví dụ:
biến:: 123
Để lấy giá trị của biến bạn dùng =this.biến → 123
Để lấy giá trị của biến nằm trong ghi chú khác, dùng =[tên note](t%C3%AAn%20note.md).biến
